### Barber Shop Management App 💈

Un'applicazione Full-Stack completa per la gestione delle prenotazioni e dei servizi di un salone di barbieri. Il progetto include un'interfaccia utente dinamica e reattiva e un sistema di gestione dati sicuro sul backend.

* * *

### 🛠️ Architettura e Tecnologie Utilizzate

*   **Frontend**: Angular, TypeScript, Bootstrap (Gestione delle viste, prenotazione appuntamenti e dashboard utente).
*   **Backend**: Java, Spring Boot (Gestione della logica di business, autenticazione e sicurezza).
*   **Database**: PostgreSQL / REST API.

* * *

### 💻 Funzionalità Principali

*   **Autenticazione**: Login e registrazione sicura per gli utenti.
*   **Prenotazione Appuntamenti**: Interfaccia interattiva per selezionare data, ora e barbiere disponibile.
*   **Catalogo Servizi**: Visualizzazione dinamica dei trattamenti offerti con relativi prezzi e tempistiche.

* * *

### 🔧 Come avviare il progetto in locale

1.  Clona il repository: `git clone https://github.com`
2.  **Backend**: Avvia l'applicazione Spring Boot tramite il tuo IDE preferito.
3.  **Frontend**: Naviga nella cartella del frontend, lancia `npm install` e successivamente `ng serve`.
